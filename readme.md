## **Mi Primer Código**

El primer código de este curso va a ser el ***"Hello world"***.

Para este codigo usaremos la libreria ***stdio.h***, la cual se añade usando ***#include<>***.

Para la funcion **main** se usara la funcion ***int*** como se muestra a continuación:


**Código**
___

```c
#include <stdio.h>

int main()
{
	printf("Hello World!\n");

	return (0);
}
